# FreshSense – IoT Based Food Spoilage Detection System

## Overview

FreshSense is an IoT-powered food spoilage detection and monitoring system designed to assess food freshness in real time using ESP32 and multiple environmental sensors.

## Features

* Real-time temperature monitoring
* Humidity monitoring
* Gas-based freshness detection
* Moisture monitoring
* LCD-based visualization
* WiFi-enabled web dashboard

## Hardware Components

* ESP32 Development Board
* DHT11 Temperature & Humidity Sensor
* MQ Gas Sensor
* Ultrasonic Sensor
* I2C LCD Display

## Software Technologies

* Embedded C++
* Arduino IDE
* HTML
* ESP32 WiFi Library

## System Architecture

Sensors → ESP32 → LCD Display + Web Dashboard

## Results

The system successfully monitors environmental conditions and helps identify potential food spoilage through real-time sensing and remote monitoring.
